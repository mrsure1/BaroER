```

🚑 핵심 기능
• 음성·이모지·메모로 증상 입력 → KTAS 중증도 자동 분류
• 공공 E-Gen 데이터 기반 실시간 수용 가능 응급실 탐색
• 수용가능 / 혼잡 / 불가 — 색상으로 한눈에 구분
• 반경 5/10/20km 즉시 조정, 지도·리스트 듀얼 뷰
• 동시 3~5곳 자동 전화 확인 (수용 여부 실시간 반영)
• KakaoNavi · Tmap · Naver Maps 내비 원터치 연동
• 구급대원 전용 모드 — 출동 자동 로그 + PDF/Excel 보고서
• Service Worker · IndexedDB 기반 오프라인 폴백

📊 기반 공공 데이터 · 표준
• 국립중앙의료원 E-Gen 응급의료정보 API
• KTAS(한국형 응급환자 분류 도구) 기준
• 보건복지부 · 공공데이터포털 공식 데이터

⚙️ 기술 스택
• Next.js 15 (App Router) · React 19 · TypeScript
• Tailwind CSS · Zustand · TanStack Query · Framer Motion
• Web Speech API (STT) · Naver Maps JS v3 · Naver Directions 5
• Supabase Auth · Next.js API Routes · Vercel Edge
• Twilio(자동 전화) · Service Worker · PWA
```
