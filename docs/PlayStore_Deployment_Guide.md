# 바로응급실 (BaroER) 구글 플레이스토어 배포 가이드

바로응급실은 모바일에 최적화된 **PWA(Progressive Web App)** 형태로 개발되었습니다. 따라서 자바(Java)나 코틀린(Kotlin)으로 안드로이드 앱을 처음부터 새로 개발할 필요 없이, 웹사이트를 안드로이드 앱으로 감싸주는 **TWA (Trusted Web Activity)** 기술을 사용하여 구글 플레이스토어에 매우 쉽고 빠르게 배포할 수 있습니다.

---

## 1. 사전 준비사항
1. **성공적인 웹 배포**: 프로젝트가 Vercel이나 서버에 배포되어 접속 가능한 `https://...` 형태의 라이브 URL이 있어야 합니다.
2. **PWA 조건 충족**: 아이콘, `manifest.json`, 서비스 워커(Service Worker)가 정상적으로 세팅되어 있어야 합니다. (현재 프로젝트에 `next-pwa`를 통해 설정되어 있습니다.)
3. **구글 플레이 개발자 계정**: 앱을 올리려면 구글 디벨로퍼 계정(등록비 1회 $25)이 필요합니다.

---

## 2. 웹앱을 안드로이드 앱 파일(.aab)로 변환하기
명령어(Bubblewrap CLI)를 직접 쳐서 만들 수도 있지만, 가장 빠르고 확실한 마이크로소프트의 **PWABuilder** 사용을 강력히 추천합니다.

1. **[PWABuilder(pwabuilder.com)](https://www.pwabuilder.com/)** 웹사이트에 접속합니다.
2. 배포된 '바로응급실'의 메인 URL(예: `https://baroer.vercel.app`)을 입력하고 **[Start]**를 클릭합니다.
3. PWABuilder가 앱의 매니페스트를 분석한 뒤 점수를 띄워줍니다. 여기서 **[Package for Android]** 버튼을 누릅니다.
4. 패키지 설정을 입력합니다:
   - **Package ID**: `com.airenaissance.baroer` (원하는 고유 ID 지정)
   - **App Name**: 바로응급실
   - **Launcher Name**: 바로응급실
5. 설정을 마치고 **[Generate]** 버튼을 누르면 `.zip` 파일이 다운로드됩니다.
6. 압축을 풀면 플레이스토어에 바로 올릴 수 있는 **`app-release.aab`** 파일(안드로이드 앱 번들)과 앱 서명 키(Key) 파일이 들어 있습니다.

---

## 3. 웹사이트 소유권 증명 (중요 ⭐️)
앱이 브라우저 주소창(상단 URL바) 없이 완벽한 전체화면 네이티브 앱처럼 실행되려면 디지털 자산 링크 인증이 필수입니다.

1. PWABuilder에서 다운받은 압축 파일 안에 **`assetlinks.json`** 이라는 파일이 있습니다.
2. 이 파일을 바로응급실 Next.js 프로젝트의 폴더 안에 넣어야 합니다.
   👉 위치: `public/.well-known/assetlinks.json`
3. 코드를 깃허브에 푸시하여 웹에 다시 배포합니다.
4. 브라우저에서 `https://내도메인/.well-known/assetlinks.json`을 쳤을 때 해당 파일이 잘 보이면 성공입니다.

---

## 4. 구글 플레이 콘솔에 등록하기
1. [구글 플레이 콘솔](https://play.google.com/console/)에 접속하여 로그인합니다.
2. **[앱 만들기]**를 클릭하고 앱 이름(바로응급실)과 기본 정보를 입력합니다.
3. 좌측 메뉴에서 **[프로덕션] -> [새 버전 만들기]**로 이동합니다.
4. 아까 PWABuilder에서 얻은 **`app-release.aab`** 파일을 마우스로 드래그하여 업로드합니다.
5. 앱 스토어에 표시될 정보들을 등록합니다.
   - 앱 아이콘
   - 스크린샷 (기존에 만든 `User_Manual.html`의 폰 목업 이미지를 스크린샷으로 쓰시면 좋습니다!)
   - 짧은 설명 및 자세한 설명
   - 개인정보처리방침 URL (필수)
6. 모든 필수 항목이 체크되면 **[출시(심사 요청)]** 버튼을 누릅니다.

보통 구글의 심사는 1일~7일 정도 소요되며, 심사가 통과되면 즉시 플레이스토어에서 '바로응급실'을 검색하여 다운로드할 수 있게 됩니다!
