# 바로응급실 (BaroER)

> 응급상황에서 환자의 상태를 입력받아 인근 수용 가능한 응급실을 신속하게 검색·안내하는 스마트폰 앱

## 프로젝트 개요

**바로응급실**은 응급 상황 발생 시 환자의 증상·상태를 빠르게 입력하고, 현재 위치 기반으로 수용 가능한 인근 응급실을 실시간으로 탐색하여 지도 및 리스트로 안내하는 모바일 애플리케이션입니다.

## 핵심 기능

| # | 기능 | 설명 |
|---|------|------|
| 1 | 사용자 인증 | 최초 1회 로그인 후 자동 로그인 유지 |
| 2 | 환자 상태 입력 | 텍스트/음성으로 증상·상태 입력 |
| 3 | 응급실 검색 | 위치 기반 인근 응급실 실시간 수용 여부 조회 |
| 4 | 지도 표시 | 수용 여부에 따른 색상 마커(🔴🟠🟢) |
| 5 | 리스트 표시 | 병원명, 거리, 소요시간, 병상수, 전화번호 등 |
| 6 | 자동 전화 확인 | 3~5곳 응급실에 동시 수용 여부 자동 전화 확인 (ON/OFF) |
| 7 | 내비게이션 연동 | 선택한 응급실로 길안내 (외부 내비 앱 연결) |
| 8 | 업무 로그 기록 | 구급대원용 상황 종료 후 처리 문서 다운로드 |

## 사용자 유형

- **일반 사용자**: 본인 또는 주변 응급환자를 위해 직접 응급실을 찾아가는 일반인
- **구급대원**: 119 구급대원 등 전문 응급 인력 (업무 로그 기능 추가 제공)

## 기술 스택 (예정)

- **프론트엔드**: React Native 또는 Flutter
- **백엔드**: Node.js / Firebase
- **데이터베이스**: Firestore / PostgreSQL
- **지도 API**: 카카오맵 API / 네이버 지도 API
- **응급실 정보**: 공공데이터포털 - 국립중앙의료원 응급의료정보 API
- **음성 인식**: Google Speech-to-Text / 카카오 음성 API
