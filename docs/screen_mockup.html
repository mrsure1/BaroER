S-001 스플래시 화면

앱 실행 시 1~2초간 표시되는 브랜딩 화면입니다. 앱 로고와 슬로건을 보여주며, 백그라운드에서 인증 토큰을 확인합니다.

  • 앱 로고 + 슬로건 표시
  • 로딩 애니메이션
  • 토큰 유효 → 자동 로그인 → 대시보드
  • 토큰 없음/만료 → 로그인 화면

S-002 로그인 화면

최초 실행 또는 토큰 만료 시 표시됩니다. 한 번 로그인하면 자동 로그인이 유지되어 이후 앱 실행 시 이 화면을 건너뜁니다.

  • 이메일/비밀번호 입력 필드
  • 자동 로그인 체크박스
  • 소셜 로그인 (카카오/구글)
  • 회원가입 / 비밀번호 찾기 링크

S-003 회원가입 화면

  • 이메일, 비밀번호, 닉네임 입력
  • 사용자 유형 선택 (일반/구급대원)
  • 구급대원 선택 시 소속기관 코드 입력 활성화
  • 가입 완료 후 자동 로그인

S-005 환자 상태 입력

응급 환자의 증상과 기본 정보를 입력하는 화면입니다. 증상 칩은 복수 선택 가능하며, 마이크 버튼으로 음성 입력도 지원합니다.

  • 증상 카테고리 칩 (복수 선택)
  • 상세 증상 텍스트 + 음성 입력 🎤
  • 성별, 연령대, 의식상태 입력
  • 검색 버튼으로 응급실 탐색 시작

S-007 리스트 모드 검색 결과

응급실 정보를 카드 리스트로 표시합니다. 각 카드의 좌측 색상 바로 수용 상태를, 우측 배지로 자동전화 확인 상태를 표시합니다.

  • 병원명, 거리, 예상 소요시간
  • 가용 병상 수 / 전체 병상 수
  • 대응 의사 유무
  • 응급실 전화번호
  • 📞 전화 / 🧭 길안내 버튼
  • 자동전화 확인 상태 배지

S-008 자동 전화 확인

3~5곳 응급실에 동시에 자동 전화를 걸어 수용 가능 여부를 확인합니다. 진행 상태가 실시간으로 업데이트됩니다.

  • ✅ 확인 완료 (수용 가능/불가)
  • 📞 통화 중 (진행 중)
  • ⏳ 대기 중
  • ❌ 확인 실패
  • ON/OFF 토글로 기능 제어
  • 결과를 지도/리스트에 즉시 반영

S-009 업무 기록 상세 (구급대원 전용)

  • 환자 상태 정보 자동 포함
  • 수용 병원 정보 (병원명, 주소, 전화)
  • 시간 타임라인 (출동→종료)
  • 총 처리 시간 자동 산출
  • 추가 메모 기록
  • 📄 PDF / 📊 Excel 다운로드
